🎮 GPT Video Game Trivia (Tkinter App)

A simple desktop trivia game built with Python and Tkinter that uses the OpenAI API to dynamically generate yes/no questions about video game history, characters, and consoles.

The game fetches real-time trivia questions, tracks your score, and provides a smooth GUI experience with themed widgets.

✨ Features

🤖 AI-generated video game trivia questions

✅ Yes/No answer system

📊 Score tracking with progress bar

🎨 Clean GUI using Tkinter + themed UI

🔄 Automatic question generation

🏁 10-question game session with final score display

🛠️ Built With

Python

Tkinter (ttk)

ttkthemes

OpenAI API

⚠️ Important Setup (API Key Required)

This project requires your own OpenAI API key to work.
